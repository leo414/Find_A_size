## 1. start
安装 Node，然后
```
$ npm install
```
因为用到了 Sass，Win 用户可能需要安装 Ruby
## 2. dev
### 2.1 开发调试
```
$ npm run dev
```
查看效果` http://127.0.0.1:8080`

### 2.2 代码检查
```
$ npm run lint
```

### 2.3 测试用例
```
$ npm run test:watch
```

### 2.4 产出资源
```
$ npm run build
```
